# Stephen Roe — Interactive Resume

**[→ View the live resume](https://stephenmroe.github.io/resume/)**

An interactive, single-page resume built as a self-contained `index.html` with
vanilla JavaScript, no framework, and no build step. Content lives in a JSON island
near the top of the file.

## Positioning

**Stephen Roe** — Principal Engineer for AI, data, and operational workflows.

I work from messy workflows and complex data toward the smallest useful product:
ship a focused vertical slice, prove it with real work, then harden and scale only
what earns the investment.

### How I build

1. Sit with the workflow and understand the people, artifacts, exceptions, and handoffs.
2. Find the smallest valuable loop where a focused intervention can prove value.
3. Ship a usable vertical slice quickly and observe it in real work.
4. Add durable architecture, controls, automation, and scale after the workflow is proven.

## Editing

To update content, edit the JSON inside the
`<script type="application/json" id="resume-data">` block in
[index.html](index.html). The renderer below it handles the page structure and interactions.
